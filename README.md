# TastyBite - Functional Food Ordering Website

This is an AI homework project.

## 5 Modules
1. Home
2. Menu
3. Cart
4. About Us
5. Contact Us

## Functionalities
- Working navigation
- Search food items
- Filter by category
- Add to cart
- Remove from cart
- Increase/decrease quantity
- Total price calculation
- Place order validation
- Contact form validation
- Responsive design

## How to Run Locally
Open `index.html` in a browser.
